-Attenuation Calculator v1.0-

Speed of Light, LLC

Summary:
This Application calculates the light loss over a fiber 
link based on user input variables.

Variables:
The user can input the following variables.

-Number of Splices
-Number of Passives
-Number of Connectors
-Length of fiber Link
-Starting Light Level

Output:
The application outputs the losses and light levels for 
the 1310nm wave and the 1550nm wave. 

Additional settings:
Although the application comes preloaded with default 
losses for wach factor, the user can edit these to their
preference. They can also restore the original defaults.
